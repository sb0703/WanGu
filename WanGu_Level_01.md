在修仙游戏的设计中，不同种族的境界名称不仅是文本替换，更代表了该种族的**修炼核心逻辑**（如：妖修体，鬼修魂，魔修欲）。

为了方便你的数值策划对齐，我将所有种族的境界与【人族】的标准境界（炼气、筑基...）进行了**横向对齐**。

---

### 1. 妖族 (The Beast/Monster Clan)
**核心逻辑：** 开启灵智 -> 锤炼肉身 -> 结出内丹 -> 褪去兽身（化形）。
妖族的特点是**肉身极强，前期修炼慢，寿命极长**。

| 人族对应 | **妖族境界** | 境界描述/状态 |
| :--- | :--- | :--- |
| **炼气** | **聚灵** (Gathering) | 野兽本能地吞吐月华，体内产生微弱妖气。 |
| **筑基** | **通智** (Sentience) | 炼化横骨，开启灵智，能口吐人言，不再是愚蠢野兽。 |
| **金丹** | **锻骨** (Forging) | 全身骨骼金属化，肉身强度暴增，体内凝聚虚丹。 |
| **元婴** | **妖丹** (Monster Core) | 虚丹化实，练出本命妖丹。妖丹不碎，生命不止。 |
| **化神** | **化形** (Transformation) | 渡过雷劫，褪去兽皮，修成人身（保留部分兽类特征如尾巴）。 |
| **炼虚** | **凝魄** (Soul Condense) | 妖魂稳固，觉醒种族天赋神通。 |
| **合体** | **返祖** (Ancestry) | 提纯血脉，向着远古神兽（龙、凤、麒麟）的形态进化。 |
| **大乘** | **天妖** (Sky Demon) | 妖气通天，肉身成圣，可硬撼天劫。 |

---

### 2. 魔族 (The Devil/Demon Clan)
**核心逻辑：** 掠夺 -> 强化魔躯 -> 凝聚魔心 -> 这是一个修“欲”和“霸”的过程。
魔族的特点是**进阶极快，容易暴毙，战斗力高**。

| 人族对应 | **魔族境界** | 境界描述/状态 |
| :--- | :--- | :--- |
| **炼气** | **聚气** (Gathering Qi) | 强行掠夺天地灵气转化为浊气（魔气）。 |
| **筑基** | **易筋** (Tendon Change) | 魔气改造凡体，痛觉丧失，力大无穷。 |
| **金丹** | **锻体** (Body Tempering) | 皮肤角质化，长出魔纹/魔角，防御力堪比法宝。 |
| **元婴** | **魔丹** (Devil Core) | 一身精血凝聚成核，此时魔族最为嗜血。 |
| **化神** | **魔婴** (Devil Infant) | 破丹生魔婴，此时可夺舍重生，极难杀死。 |
| **炼虚** | **离识** (Separation) | 魔识离体，可分化万千魔头去污染他人。 |
| **合体** | **合体** (Fusion) | 魔婴与魔躯完美融合，不死不灭之身初成。 |
| **大乘** | **大天魔** (Archdevil) | 化身自在天魔，以众生为食。 |

---

### 3. 鬼族/魂修 (The Ghost/Specter Clan)
**核心逻辑：** 聚阴气 -> 凝实魂体 -> 修炼出实体 -> 不入轮回。
鬼族的特点是**物理免疫，惧怕雷/火，诡异莫测**。

| 人族对应 | **鬼族境界** | 境界描述/状态 |
| :--- | :--- | :--- |
| **炼气** | **游魂** (Wandering Soul) | 意识混沌，随时可能消散，仅凭本能吸食阴气。 |
| **筑基** | **鬼卒** (Ghost Pawn) | 意识觉醒，魂体凝实，不再畏惧微弱的阳光。 |
| **金丹** | **鬼将** (Ghost General) | 魂体如实质，可御物杀人，统领一方孤魂。 |
| **元婴** | **鬼王** (Ghost King) | 凝聚“鬼丹/魂珠”，在鬼蜮中是一方霸主。 |
| **化神** | **鬼仙** (Ghost Immortal) | 脱去死气，转阴为阳，重塑类似活人的肉身。 |
| **炼虚** | **摄魂** (Soul Taker) | 掌握灵魂法则，一眼可勾人魂魄。 |
| **合体** | **渡魂** (Ferryman) | 游走于阴阳两界，甚至可以欺骗生死簿。 |
| **大乘** | **鬼帝** (Ghost Emperor) | 幽冥之主，自成一界。 |

### 4. 灵族 (The Spirit/Elemental Clan)
**核心逻辑：** 只有草木/石头/器物成精才叫灵族。
过程：死物 -> 开启灵智 -> 幻化形体。

| 人族对应 | **灵族境界** | 境界描述/状态 |
| :--- | :--- | :--- |
| **炼气** | **启灵** (Awakening) | 一块石头或一株草忽然有了模糊的意识。 |
| **筑基** | **感知** (Sensing) | 根须/本体延伸，能感知周围环境并趋吉避凶。 |
| **金丹** | **幻形** (Illusion) | 可以制造幻象迷惑敌人，本体依然脆弱。 |
| **元婴** | **聚形** (Solidification) | 将灵气凝聚成实体（通常是童子/少女形象）。 |
| **化神** | **融合** (Fusion) | 本体与化身融合，不再有弱点。 |
| **炼虚** | **自然** (Nature) | 举手投足引动天地元素（如火灵族引动火山）。 |

### 5. 古神/巫族 (The Ancient/Shaman Clan)
**核心逻辑：** 不修灵气，只修肉身血气。参考《盘古》传说。
特点：**没有花哨的法术，一拳破万法**。

| 人族对应 | **巫族境界** | 境界描述/状态 |
| :--- | :--- | :--- |
| **炼气** | **蛮力** (Brute Force) | 单臂千斤，铜皮。 |
| **筑基** | **铁骨** (Iron Bone) | 刀枪不入，骨骼如精铁。 |
| **金丹** | **沸血** (Boiling Blood) | 气血如龙，受伤后战力飙升（狂战士）。 |
| **元婴** | **战纹** (War Paint) | 皮肤表面浮现先天图腾，获得法则加持。 |
| **化神** | **山海** (Mountain & Sea)| 法天象地，身躯可化作百丈巨人。 |
| **炼虚** | **破空** (Shattering) | 肉身力量足以粉碎虚空。 |
| **合体** | **星辰** (Star) | 可以在虚空中生存，以星光为食。 |

### 给策划的建议 (Implementation Tips)

1.  **UI显示上的差异化：**
    *   人族玩家看到的界面是：`修为: 筑基中期`
    *   如果你开放了妖族路线，界面应该自动变为：`修为: 通智中期`
    *   这种细节会让玩家觉得“这游戏做得真细”。

2.  **突破材料的差异化：**
    *   人族筑基用 **【筑基丹】**。
    *   妖族通智可能需要 **【帝流浆】**（月光精华）。
    *   魔族易筋可能需要 **【生灵血】**。
    *   鬼族可能需要 **【养魂香】**。
    *   *设计点：* 这鼓励了不同种族的玩家在拍卖行进行交易。

3.  **称谓系统 (Titles)：**
    *   境界是力量层级，**称谓**是社会地位。
    *   **妖族称谓：** 小妖 -> 大妖 -> 妖王 -> 妖皇 -> 妖圣。
    *   **魔族称谓：** 魔兵 -> 魔将 -> 魔帅 -> 魔君 -> 魔尊。
    *   玩家在P2阶段建立宗门时，这些称谓可以用作NPC的职级。
