在《万古墨境》的设计理念中，**“疗伤”本身就是一次冒险**。
如果玩家身中剧毒，他不应该只是点一下“使用解毒草”，而可能需要去极寒之地寻找冰蟾，或者去求助一位性格古怪的神医。

我们将消除 Debuff 的方式分为四大类：**丹药医术（常规）**、**天地机缘（探索）**、**红尘因果（社交）**、**旁门左道（代价）**。

以下是详细的设计表格：

### 一、 丹药与医术 (Alchemy & Medical)
*最正统的解决方式，依赖【炼丹系统】或【货币能力】。*

| 针对症状 | 疗法名称 | 具体操作 / 消耗 | 玩法深度 / 备注 |
| :--- | :--- | :--- | :--- |
| **灵气逆流**<br>(内伤) | **推宫过血** | 寻找【医师】类NPC，花费灵石进行推拿。 | 便宜但耗时，需消耗游戏内时间1-3个月。 |
| **经脉淤塞**<br>(废穴) | **炼制通脉丹** | 需收集：三阶妖兽内丹 + 通天草。自行炼制或拍卖行购买。 | 强推炼丹玩法，让玩家感受“求药难”。 |
| **肢体残缺**<br>(断臂) | **生肌造骨** | 服用六品以上【生骨融血丹】。 | 极度昂贵，通常只有大后期才能获得，前期断臂建议走“义肢”路线。 |
| **丹毒攻心**<br>(抗药性) | **洗髓金针** | 寻找隐世神医（奇遇NPC），完成其怪异请求（如：帮他试毒）后施针。 | 这是一个小型支线任务链。 |
| **神识撕裂**<br>(晕眩) | **温养识海** | 需长期服用【养魂木】泡的茶，或在洞府点燃【安神香】挂机。 | 将“挂机”赋予疗伤的意义。 |

---

### 二、 天地与机缘 (Exploration & World)
*利用环境特性来消除 Debuff，强调【地图探索】和【五行生克】。*

| 针对症状 | 疗法名称 | 具体操作 / 消耗 | 玩法深度 / 备注 |
| :--- | :--- | :--- | :--- |
| **火毒/热症** | **寒潭淬体** | 需前往地图北部的【极寒冰原】，找到“万年寒潭”节点进行打坐。 | 只有特定地点有效，且打坐时可能遭遇冰系妖兽袭击。 |
| **心魔滋生** | **佛光普照** | 前往【古刹/寺庙】场景，聆听高僧诵经或撞钟。 | 需捐赠香火钱（大量凡俗金银），通过净化心灵消除Debuff。 |
| **寿元折损** | **吞噬生机** | 寻找天材地宝【寿元果】（极低概率掉落）或寻找【长生泉】。 | 典型的Roguelike寻宝玩法，全看脸。 |
| **虚空流放** | **寻找界碑** | 在未知地图中探索，寻找【空间薄弱点】，使用“破界符”逃脱。 | 这是一个迷宫逃脱小游戏，失败可能死在里面。 |
| **天道厌弃**<br>(霉运) | **功德金身** | 前往凡人城市处理【水患/瘟疫】事件，积累大量功德值。 | 强迫玩家进行P2阶段的“入世”玩法。 |

---

### 三、 红尘与因果 (Social & Relationships)
*利用【NPC关系网】解决问题，有时候人脉比修为更有用。*

| 针对症状 | 疗法名称 | 具体操作 / 消耗 | 玩法深度 / 备注 |
| :--- | :--- | :--- | :--- |
| **道心破碎**<br>(忘招) | **高人传道** | 寻找好感度极高（崇拜/生死之交）的高阶NPC，请求其【讲道】。 | 需要前期长期投资NPC好感度，此时才能获得回报。 |
| **仇家锁定** | **寻求庇护** | 躲入【宗门禁地】或花费重金寻求一位【大乘期老祖】的庇护。 | 玩家需缴纳巨额“保护费”，或者成为老祖的“棋子”（接强制任务）。 |
| **性格扭曲** | **红尘炼心** | 与一位【心思纯良】的异性NPC结为道侣，并进行长期互动（双修/游历）。 | 也就是“用爱感化”，慢慢把性格标签洗回来。 |
| **树倒猢狲散** | **重金招募** | 散财童子战术。发布【悬赏令】，以双倍俸禄招募新弟子/随从。 | 只要有钱，忠诚度是可以买回来的。 |
| **阴阳逆转** | **双修调和** | 寻找一位特殊体质（如纯阴/纯阳）的道侣，通过【阴阳调和】慢慢恢复。 | 这是一个漫长的过程，甚至可能在这个过程中爱上新性别... |

### 四、 旁门与左道 (Forbidden Arts & Trade-off)
*不走寻常路。我不治了，我直接把坏的切了换个新的，或者以毒攻毒。*

| 针对症状 | 疗法名称 | 具体操作 / 消耗 | 玩法深度 / 备注 |
| :--- | :--- | :--- | :--- |
| **肢体残缺** | **偃甲义肢** | **炼器玩法联动：** 用玄铁和妖骨打造一条【机关臂】装上。 | **转化玩法：** 手臂无法拿剑，但可以发射暗器/掌心雷。 |
| **肢体残缺** | **妖兽移植** | **猎杀玩法联动：** 砍下一只麒麟的手臂接在自己身上。 | **转化玩法：** 获得【麒麟臂】词条，力量大增，但每回合有概率失控攻击队友。 |
| **识海寄生**<br>(天魔) | **共生契约** | 不驱逐天魔，反而与其【签订契约】，主动献祭部分神识。 | **转化玩法：** 获得【天魔化】变身技能，但永久扣除悟性，且容易被正道追杀。 |
| **肉身损毁** | **夺舍重生** | 放弃治疗，直接元婴出窍，寻找一个凡人肉身夺舍。 | **重开玩法：** 继承记忆和功法，但身体资质（根骨/样貌）随机重置。 |
| **丹毒攻心** | **毒功转化** | 转修魔道功法《万毒噬心诀》，将体内毒素转化为修为。 | **流派切换：** 从此变成毒修，以前的正道功法威力减半。 |

### 给策划的执行建议

在开发P1/P2阶段时，不需要把所有疗法都做成复杂的系统，可以用**“标签+脚本”**的方式简化实现：

1.  **Tag检测：** NPC医生检测到玩家有 `Has_Buff('断臂')`。
2.  **触发对话：** “少侠这手断得整齐啊，老夫这里有条刚切下来的虎臂，只要5000灵石，你要不要接上？”
3.  **分支选择：**
    *   [A. 接上虎臂] -> 移除Debuff '断臂' -> 获得Buff '虎臂(力量+20, 魅力-50)'
    *   [B. 只要人手] -> 提示“老夫没货，去隔壁村看看吧”

这样设计，**消除Debuff的过程本身就变成了游戏最好玩的故事线之一**。
