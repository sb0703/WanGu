文字修仙游戏的登录与创角界面，是玩家对这个“世界”的第一印象。它不需要华丽的粒子特效，但需要**极强的仪式感**。

设计核心理念：**“一念起，万水千山；一念灭，沧海桑田。”**

---

# 一、 登录界面：【混沌初开】(Login Screen)

**视觉风格：** 纯黑背景。屏幕中央只有一行行如水墨晕开般的文字缓缓浮现，背景伴有极其轻微的古琴或风铃声。

### 1. 界面布局 (UI Layout)

```text
+--------------------------------------------------+
|  （背景：漆黑一片，偶尔有流光/文字碎片划过）       |
|                                                  |
|              万 古 墨 境                         |
|           ----------------                       |
|              红  尘  渡                          |
|                                                  |
|        " 仙 路 尽 头 谁 为 峰 "                  |
|        " 一 见 无 始 道 成 空 "                  |
|                                                  |
|                                                  |
|            [ 踏 入 仙 途 ]  (开始游戏/注册)      |
|            [ 再 续 前 缘 ]  (读取存档/登录)      |
|                                                  |
|                                                  |
|     [天道宝鉴(设置)]      [红尘轶事(公告)]       |
|                                                  |
|          v0.1.0_Alpha | ID: [未连接]             |
+--------------------------------------------------+
```

### 2. 交互逻辑

*   **踏入仙途 (New Game)：**
    *   点击后，屏幕文字如烟雾般散去。
    *   **动画文本：** “你推开了尘封已久的青铜门……” -> 跳转至【创角界面】。
*   **再续前缘 (Continue/Login)：**
    *   **未登录状态：** 弹出手机号/账号登录框（样式为“神识连接”）。
    *   **已登录状态：** 显示本地存档列表（如：*道号：韩跑跑 | 境界：筑基后期 | 停留地：乱星海*）。
*   **背景彩蛋：**
    *   屏幕上的两句诗（Slogan）每次打开游戏随机更换。
    *   例如：“天地不仁，以万物为刍狗”、“顺为凡，逆则仙”。

# 二、 创角界面：【轮回转生】(Character Creation)

这不是简单的捏脸，而是**“投胎”**。玩家需要在这里决定自己的**出身、灵根、命格**。
界面采用**分步式引导（Wizard）**，每一步都像是在生死簿上书写。

### 第一步：【名讳与皮囊】(Identity)

```text
> 冥冥之中，你听到了唤魂的声音……
> "来者何人？"

姓：[ 输入 ]  名：[ 输入 ]   [🎲 随机(掷骰子)]
(随机库包含有修仙味的名字，如：叶凡、厉飞雨、南宫婉)

性别：[男] [女]
(影响部分功法修习，如《葵花宝典》或《素女经》)

相貌标签 (影响魅力值与NPC初见印象)：
[ ] 清秀 (魅力+5)
[ ] 粗犷 (威慑+5)
[ ] 平平无奇 (存在感低，不易被追杀)
[ ] 仙姿绝色 (魅力+20，极易触发桃花劫)

[ 下一步：测灵根 ]
```

---

### 第二步：【灵根与资质】(Spirit Roots - Roguelike Reroll)

这是最刺激的环节。玩家点击“投胎”按钮，随机生成属性。
**设计重点：** 提供**“逆天改命”**（重随）功能。

```text
> 触摸测灵碑，你的掌心泛起光芒……

【 灵 根 资 质 】
--------------------------------
金：|||||||||| (80)  [天品]
木：||         (20)  [杂品]
水：|          (10)  [废品]
火：           (0)   [无]
土：|||        (30)  [凡品]
--------------------------------
判定：【金系天灵根】
描述：你是天生的剑修苗子，修炼金系功法速度+200%。

【 先 天 属 性 】
--------------------------------
根骨：25 (血量/防御成长)
悟性：40 (经验获取速度)
机缘：88 (甚至走路都能踢到宝)
家世：10 (初始灵石/开局道具)
--------------------------------

[ ⚡ 逆天改命 (重随 Reroll) ] 
(剩余次数：3 / 看广告增加)

[ 下一步：定命格 ]
```

---

### 第三步：【命格选择】(Traits Selection)

系统随机给出 3-5 个词条（有正面也有负面），玩家必须**选择 3 个**作为初始天赋。

```text
> 轮回通道中，几道流光钻入了你的体内……
> 请选择 3 项命格融入灵魂：

[X] 穿越者 (金)
    描述：你来自异世界。悟性+20，初始获得一本《小学语文》。
[ ] 独臂大侠 (红 - 负面)
    描述：出生时少一条手臂。无法装备副手，攻击力-10%。
[X] 伴生异火 (紫)
    描述：出生时体内封印着一团青莲地心火。火系伤害+30%。
[ ] 绝户手 (蓝)
    描述：对异性造成的伤害+20%。
[X] 扫把星 (灰)
    描述：幸运值-10，但更容易遇到危险（高风险高回报）。

[ 确 定 ]
```

# 三、 特殊状态处理

### 1. 正在排队/加载中
不要用转圈圈。
**文案：**
> “正在穿越空间乱流...”
> “正在重塑肉身...”
> “正在与天道建立连接...”

### 2. 名字重复
**文案：**
> “此方世界已有大能名为[叶傲天]，为避因果，请道友更名。”

### 3. 断线重连
**文案：**
> “神识连接中断，正在尝试重新感应肉身...”

# 四、 创角策略设计建议 (Gameplay Logic)

1.  **彩蛋名字：** 如果玩家输入特定名字（如“韩立”、“王林”），虽然不能真的让他无敌，但可以给一个隐藏的小彩蛋（如：初始多给一个【神秘小瓶】或【铁剑】），增加传播话题性。
2.  **家世的影响：**
    *   *家世 0-20 (孤儿/乞丐)：* 开局只有破碗，但可能获得【坚韧】命格（防御+5%）。
    *   *家世 80-100 (皇族/世家)：* 开局带【侍女】、【玉佩】和大量灵石，但可能获得【纨绔子弟】命格（修炼速度-10%）。
3.  **灵根的视觉化：** 在P0阶段，可以用简单的 `=======` 进度条表示；在P2阶段，可以做一个五角星雷达图，看起来更专业。

这个设计旨在让玩家在还没开始打怪前，就先玩了5分钟的“投胎模拟器”，这是文字修仙游戏留存率极高的关键点。
